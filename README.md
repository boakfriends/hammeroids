# Hammeroids

Description:
A little game with a JS frontend and Ruby backend

Author: Jason Porter (and Dan Goodwin)


## Setup
Requires Ruby 2.6.1

Run to install gems
```
bundle
```

Run to install other local dependencies.

```
brew bundle
```

Copy `.env.sample` to `.env` and set values.
